# ARMory

Система управления документацией проектов.

## Возможности

- **Проекты** — создавайте проекты, разбивайте их на разделы и группы.
- **Файлы, заметки и ссылки** — храните файлы, текстовые заметки и ссылки в одном месте.
- **Замена файлов** — обновляйте файл, сохраняя его название и историю версий.
- **Контекстное меню файлов** — правый клик по элементу: копировать ссылку, предпросмотр, открыть в Alexandrite, изменить, скачать, удалить.
- **Сортировка** — меняйте порядок групп и элементов внутри групп перетаскиванием.
- **Пагинация** — удобная навигация по длинным спискам проектов с prev/next, окном страниц и многоточием.
- **Поиск** — быстрый поиск по проектам и содержимому.
- **Alexandrite** — встроенное Markdown-хранилище знаний с двухпанельным интерфейсом.
  - Локальное редактирование файлов и папок.
  - Read-only просмотр заметок с Яндекс.Диска.
  - Центрированный предпросмотр текста, Markdown и изображений.
  - Редактирование офисных документов через Collabora Online.
  - Сохранение состояния дерева между операциями.
- **Яндекс.Диск** — синхронизация базы данных, загруженных файлов и Alexandrite с облаком.
- **Архивные бэкапы** — создание и восстановление архивов `.tar.gz` для ARMory и Alexandrite.
- **Планировщик задач** — задачи с датами, повторениями, приоритетами и напоминаниями.
- **Календарь** — визуальное представление задач и событий по месяцам.
- **Глоссарий** — база терминов с темами, подтемами, импортом и экспортом в `.xlsx`.
- **PocketBase** — расширение для коллективной работы: комментарии, задачи и заметки по проектам.
- **Адаптивный UI** — Bootstrap 5, сворачиваемые сайдбары, сохранение состояния интерфейса.

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd ARMory

# 2. Скопировать и заполнить .env
cp .env.example .env

# 3. Запустить через Docker Compose
docker compose up -d --build

# 4. Открыть в браузере
http://localhost:8000
```

Для включения PocketBase установите `POCKETBASE_ENABLED=true` в `.env`. Локально без Docker запустите `./scripts/run_pocketbase.sh` после распаковки бинарника в `./pocketbase`.
