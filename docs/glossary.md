# Глоссарий

**Глоссарий** — терминологический словарь проекта. Он помогает фиксировать термины и их значения, чтобы все участники проекта использовали единую терминологию.

## Возможности

- Добавление терминов с кратким и развёрнутым определением.
- Темы и подтемы — термины можно группировать по темам, а внутри тем — по подтемам.
- Фильтрация терминов по теме или подтеме через левый сайдбар.
- Поиск по термину и определению.
- Карточки-аккордеоны: в свёрнутом виде показывается только название термина, тема и подтема; краткое и развёрнутое определения открываются по клику.
- Визуальное выделение краткого и развёрнутого определений внутри раскрытой карточки.
- Кнопки редактирования и удаления термина прямо в карточке.
- Числовая пагинация.

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/glossary` | Список терминов с пагинацией, поиском и фильтром по теме/подтеме |
| GET | `/api/glossary/count` | Количество терминов с учётом фильтров |
| POST | `/api/glossary` | Создать термин |
| GET | `/api/glossary/{id}` | Получить термин |
| PATCH | `/api/glossary/{id}` | Обновить термин |
| DELETE | `/api/glossary/{id}` | Удалить термин |
| GET | `/api/glossary/topics` | Список тем с подтемами и счётчиком терминов |
| POST | `/api/glossary/topics` | Создать тему |
| PATCH | `/api/glossary/topics/{id}` | Обновить тему |
| DELETE | `/api/glossary/topics/{id}` | Удалить тему (термины становятся «без темы») |
| GET | `/api/glossary/subtopics` | Список подтем |
| POST | `/api/glossary/subtopics` | Создать подтему |
| PATCH | `/api/glossary/subtopics/{id}` | Обновить подтему |
| DELETE | `/api/glossary/subtopics/{id}` | Удалить подтему |
