# Канбан

Канбан-доска доступна внутри каждого проекта по адресу `/projects/<id>/kanban`.
Общая доска по всем проектам — `/kanban`.

## Возможности

- **Колонки** — настраиваемые статусы задач с цветом и порядком.
- **Задачи** — описание, приоритет, дедлайн, ответственный, теги. Название необязательно: без него задача отображается как «Заявка #N».
- **Закрытие задач** — чекбокс «Закрыто» в форме задачи. Закрытая задача становится неактивной: сбрасывается дедлайн, задача уходит вниз колонки и в конец списков (сортировка по активности и дате создания). При переоткрытии задача возвращается в начало колонки.
- **Вложения** — ссылки, файлы, git-репозитории. У вложений-ссылок есть зелёная кнопка копирования ссылки. Предпросмотр файлов и редактирование в Collabora Online открываются в новой вкладке.
- **Drag-and-drop** — перемещение задач между колонками.
- **Фильтры** — по приоритету, ответственному, тегу, дедлайну и дате создания.
- **Импорт/экспорт** — JSON для переноса доски между проектами.
- **Прямые ссылки** — `/projects/<id>/kanban?task=<task_id>` подсвечивает задачу на доске.

## Создание задачи из колонки

Кнопка «Добавить задачу» внизу каждой колонки открывает форму с подставленным статусом этой колонки:

- В канбане проекта проект уже известен — статус колонки сразу выбран в форме.
- В общем канбане (`/kanban`) проект не заполняется автоматически: выберите проект вручную, и статус подставится по имени колонки (если в проекте есть статус с таким именем).

## Список задач (ToDo-список)

Кнопка «ToDo-список» на доске открывает таблицу всех задач: `/projects/<id>/tasks` — по проекту, `/tasks` — по всем проектам.

- Все поля задачи в колонках, сортировка кликом по заголовку.
- Фильтры (оформлены как на канбане): поиск, проект, статус, приоритет, ответственный, список, состояние (открыта/закрыта), теги.
- Таблица на всю ширину экрана, горизонтальная прокрутка сверху и снизу; фиксированная высота строк — длинное описание обрезается многоточием.
- **Сохранить список** — выгрузка отфильтрованных задач в текстовый вид (копирование или скачивание `.txt`): выбор колонок и формата — нумерованный список (по умолчанию), чек-лист ToDo, Markdown-таблица, одна строка — одна задача. Колонки, по которым выставлены фильтры, снимаются с выбора автоматически.
- **Экспорт в JSON/CSV** — включает название проекта; даты и метка времени в имени файла — по московскому времени (UTC+3).

## Импорт задач

Кнопка «Импорт задач» на странице списка задач создаёт задачи из текстового списка:

- Каждый импортированный список помечается своим названием (колонка «Список») — по нему можно фильтровать и сортировать.
- Текст каждого пункта попадает в «Описание» задачи, название не заполняется.
- Доступны массовые настройки (дедлайн, приоритет, ответственный, теги) и общие вложения — они применяются к каждой задаче списка.

## Прямые ссылки на задачи

Чтобы поделиться конкретной задачей, скопируй ссылку вида:

```text
/projects/3/kanban?task=42
```

При открытии такой ссылки задача подсвечивается на доске. Первый клик по подсвеченной карточке только снимает подсветку; повторный клик открывает задачу как обычно.
