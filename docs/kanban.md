# Канбан

Канбан-доска доступна внутри каждого проекта по адресу `/projects/<id>/kanban`.

## Возможности

- **Колонки** — настраиваемые статусы задач с цветом и порядком.
- **Задачи** — название, описание, приоритет, дедлайн, ответственный, теги.
- **Вложения** — ссылки, файлы, git-репозитории.
- **Drag-and-drop** — перемещение задач между колонками.
- **Фильтры** — по приоритету, ответственному, тегу, дедлайну и дате создания.
- **Импорт/экспорт** — JSON для переноса доски между проектами.
- **Прямые ссылки** — `/projects/<id>/kanban?task=<task_id>` подсвечивает задачу на доске.

## Прямые ссылки на задачи

Чтобы поделиться конкретной задачей, скопируй ссылку вида:

```text
/projects/3/kanban?task=42
```

При открытии такой ссылки задача подсвечивается на доске. Первый клик по подсвеченной карточке только снимает подсветку; повторный клик открывает задачу как обычно.
